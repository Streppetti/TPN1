### Trabajo Practico N°1 - Réplica Página web Nave con fin educativo

El objetivo de este trabajo práctico es recrear la página web de Nave Negocios, utilizando lo aprendido hasta el momento en la cursada.

***En este es importante:***

- El uso correcto y la semántica del HTML.
- El uso correcto de CCS (Flexbox)
- Buena identación, claridad y organización del código
- La utilización de media queries para un lograr un diseño adaptable en distintos dispositivos
- Un responsive design desde 320px hasta 2000px de ancho en el cual se mantenga una buena visualización
- Fidelidad visual al diseño original de Nave

##### Planificación
Para trabajar de forma óptima y organizada, lo planifique de la siguiente manera:

1- Ver en detalle la web original

Ver estructuras, aspectos, medidas, divisores.

2- Realizar croquis del diseño de la web (Se realizo con “Excalidraw”)

Dividí la web por secciones para trabajar en bloques, marcando medidas, estructuras para organizar mejor a la hora de definir estilos, posiciones de cada elemento.

3- Descargar Material necesario (Imágenes, Fuentes, Iconos)

4- Realizar código etapa por etapa

HTML —> CSS (Lo realice por las secciones previamente definida).

5- Realizar “Responsive Design” y “Media Queries”

6- Limpiar código y terminación

7) Realizar “Readme”

##### Estructura del HTML

- header: Barra de Navegación
- main: Esta compuesto por sections
	- Carousel (Carrusel de Fotos)
	- Welcome (Bloque de Informacion principal de Biernvenida)
	- Pay (Bloques con los medios de pagos)
- footer	(Links externes y contacto)

El objetivo de esta estructura fue minimizar tantos elementos a definir estilos y trabajarlos por “cajas” para facilitar el orden de la web.
